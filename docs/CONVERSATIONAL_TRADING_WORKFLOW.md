# Conversational Trading Workflow

ZenInvest supports a conversational interface for reviewing, confirming, and
executing trading actions.

## Supported Interaction Shapes

- review a ticker before trading
- ask for a direct buy or sell
- confirm or reject a proposed action
- cancel pending orders
- inspect related research context

## Workflow Characteristics

- actions are explicit and auditable
- confirmation is required for sensitive or higher-risk flows
- session context is preserved across multi-turn conversations
- deterministic checks remain authoritative even when the interface is natural
  language driven

The public mirror documents the workflow at a product level. Operator-specific
runbooks and deployment procedures are intentionally omitted.
