# Investment Agent — CV Impact

How this project strengthens my professional narrative and fills specific gaps in my CV.

## The Story It Tells

The Investment Agent is a personal quant system — an autonomous multi-LLM trading agent I designed, built, and deployed end-to-end. It demonstrates that my AI leadership experience isn't just managerial: I can architect, code, deploy, and govern a complex AI system from scratch.

At L&G, I lead teams and own strategy. This project proves I also *do the work* — and at a level of sophistication (multi-model orchestration, agentic AI, quantitative finance) that goes well beyond typical side projects.

## Key CV Differentiators

### 1. Multi-LLM System Architecture (rare, high-signal)

Most people use one LLM. I orchestrate three (Claude, GPT-4o, Gemini) in an adversarial pipeline where they challenge each other's reasoning. This is genuinely novel — the "investment committee" pattern with independent AI voices, consensus logic, and deterministic veto gates.

**CV angle:** "Designed and deployed a multi-LLM adversarial pipeline (Claude + GPT-4o + Gemini) with consensus-based decision-making and deterministic risk veto — demonstrating advanced AI orchestration beyond single-model applications."

### 2. AI Governance as Architecture (not just policy)

My CV already shows governance leadership at L&G (model risk, GDPR, responsible AI). This project proves I can *implement* governance as code: 11 hard rules in deterministic Python that no LLM can override, a drawdown state machine, cost degradation chains, and a comprehensive audit trail across 12 database tables.

**CV angle:** "Implemented defense-in-depth AI governance: deterministic risk rules with architectural isolation from LLM outputs, automated cost controls with graceful degradation, and full decision audit trail — bridging governance policy with production engineering."

### 3. Agentic AI (cutting edge, beyond RAG)

My L&G experience shows RAG architecture (Azure OpenAI, medical document summarisation). The Investment Agent goes further: agentic research where LLMs autonomously use tools (web search, SEC EDGAR, news) mid-reasoning, with shared pipeline budgets and per-member call caps. This is the direction the industry is heading.

**CV angle:** "Built agentic AI capabilities with tool-use loops (web search, SEC EDGAR, news) enabling LLM committee members to independently research during decision-making — advancing beyond retrieval-augmented generation to autonomous reasoning."

### 4. Full-Stack AI Product (end-to-end)

Not just a model or notebook — a complete product: Python backend, FastAPI REST API + SSE streaming, React dashboard (11-page authenticated surface, ZENOUZ.ai brand), Docker Compose deployment, APScheduler, SQLite with Alembic migrations, Slack/Email notifications. Deployed on VPS.

**CV angle:** "Delivered a full-stack AI product: Python pipeline, FastAPI API, React dashboard, Docker deployment, real-time notifications — demonstrating end-to-end product delivery from ML to production."

### 5. Quantitative Finance Domain (broadens sector reach)

My CV is strong in insurance/financial services. This project adds quantitative finance: technical indicators, fundamental analysis, backtesting with walk-forward validation, portfolio risk management, order execution with stop-loss and trailing stops. Relevant for asset management, hedge fund, or fintech roles.

**CV angle:** "Applied quantitative methods (momentum, mean reversion, factor strategies) with backtesting, walk-forward validation, and automated portfolio risk management — demonstrating fluency in quantitative finance alongside insurance and data science."

### 6. Evidence-Based Engineering Culture

The project embodies the "measure before you build" philosophy: 703 tests, backtesting with promotion reports (fail-closed — weak evidence blocks deployment), experiment reproducibility (seed + config), and a sophistication roadmap that gates features on data availability. This mirrors how I run teams at L&G.

**CV angle:** "Applied evidence-based engineering: 703 automated tests, walk-forward backtesting with quantitative promotion gates, and a data-gated sophistication roadmap — the same discipline I bring to enterprise AI delivery."
